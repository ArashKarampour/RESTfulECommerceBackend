# Spring Boot Advanced REST APIs with Authentication and Authorization

You can clone this project and run it on you local machine.
```sh
git clone git@github.com:ArashKarampour/RESTfulECommerceBackend.git
```
## What is done in this project?

1- Implementing RESTful APIs for an e-commerce backend using Spring Boot (CRUD Operation, DTOs (Data serilization and deserilization), Mappers).

2- Implementing authentication and authorization using Spring Security.

3- Using JWT (JSON Web Tokens) for producing security token as well as refresh token.

4- Implementing role-based access control to restrict access to certain endpoints based on user roles.

5- Build a shopping cart system with full checkout functionality

6- Validating incoming requests and handle errors with custom logic

7- Structuring the application using clean, maintainable architecture

8- Testing the APIs using Postman or similar tools to ensure they work as expected.

## What is under work?

- Implemneting a Stripe payment gateway for processing payments securely.

- Implementing email notifications for order confirmations, password resets, and other user interactions.

- Unit and integration testing to ensure the reliability and stability of the application.
